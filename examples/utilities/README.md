# Utility Examples

This directory contains utility examples and helper guides for working with NexureJS.

## 📁 Contents

### 🔧 **Development Utilities**
- **[Fallback Demo](fallback-demo.js)** - Demonstrates automatic fallback to JavaScript when native modules are unavailable
- **[Worker Task](worker-task.js)** - Example worker thread implementation for CPU-intensive tasks

### 📖 **Guides & Documentation**
- **[Import Without Extensions](import-without-extensions.md)** - Complete guide for using imports without file extensions

## 🚀 Quick Start

### Running Utilities
```bash
# Test fallback behavior
node examples/utilities/fallback-demo.js

# Test with forced JavaScript fallback
node examples/utilities/fallback-demo.js --force-js

# Test worker task
node examples/utilities/worker-task.js
```
